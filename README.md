# Text101

Text101 is a text-based adventure game where the user reads the story and
choose from the available options. The story will proceed depending on the
choices of the user.

## The Story
The prisoner start in his/her cell. The only notable items are bed sheets, a
mirror on the wall and a locked door.

### Cell + Corridor Scenes
The first two scenes follow that of the Udemy course. More details can be found
in the `Text101GDD.pdf` file.
